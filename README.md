Projeto Train.Info

Autores:

André Erci Freitag

O projeto:

Durante o ano de 2025, a turma de Desenvolvimento de Sistemas 24M6 do Terceiro ano da escola SESI de Referência foi desafiada a montar e implementar um site de monitoramento e suporte para uma empresa de trens, assim surgiu o Train.Info, um site projetado e desenvolvido pelos desenvolvedores:

André Erci Freitag

Helena Da Silva Cani

Rafael Franklyn de Farias

Iago König Kleestad

O presente projeto se desenvolveu em um site com sistema de login para funcionários e controle dos mesmos, onde é possível não apenas o monitoramento dos trens, mas tambem a garantia de segurança dos mesmos e dos maquinistas que operam essas máquinas.

Foram usadas das mais diversas tecnologias, como um sistema de banco de dados SQL para controle das imformações dos funcionarios, trens, manutenções dos trens, rotas e relatorios graficos de nossos desempenhos como empresa.

O site foi programado em PHP e JavaScript, para que ele possa atender a todas as demandas pedidas pela empresa, se tornando um site visivelmente agradavel, responsivo e claro funcional.

Pela demanda das diversas funcionalidades do site como monitoramente em tempo real dos trens, relatorio dos maquinistas sobre seus problemas ou problemas de trens, além de notificações e alertas em caso de crises, o site atualmente conta com uma conexão em tempo real por IoT para que tudo possa estar amplamente conectado.

Com tudo isso sendo necessário, tivemos um repositorio vasto e muito bem desenvolvido, onde cada um dos desenvolvedores ficou responsavel por uma parte expecifica do site, assim separando as tarefas e as reunindo para a criação do site como um todo, os documentos por questõa de boas praticas estão separados por tipo, e nomeados com suas respectivas funções.
